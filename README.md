# mohammadgheisari1994.github.io

Personal portfolio and CV website for GitHub Pages.

## Local preview
From the repository root:
```bash
python3 -m http.server
```
Then open the printed URL in your browser.

## Structure
- `index.html`: main entry point
- `assets/css/base.css`: global styles
- `src/core`: app shell, routing, effects
- `src/sections`: modular section components
